== Description ==

Best analytics plugin for having analytics installed on live sites and dev sites without showing up on both.

Major features in WP Best Analytics include:

* Includes meta field option for verifying with Google Search Console and Bing Webmaster Tools
* Define dev site URLs where you don't want the analytics code to show. 

== Installation ==

Upload WP Best Analytics folder to your wp-content/plugins directory, then Activate it. Settings for the plugin will appear in the standard WordPress "Settings" menu.
